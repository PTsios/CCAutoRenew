# CCAutoRenew — правила для Claude

## Автосинхронизация прогресса

Перед завершением сессии, если была проделана значимая работа — напомни пользователю: "Запустить `/sync` чтобы сохранить прогресс?"

Также предложи `/sync` если пользователь говорит: "стоп", "на сегодня всё", "продолжим позже", "сохрани прогресс", "хватит".
